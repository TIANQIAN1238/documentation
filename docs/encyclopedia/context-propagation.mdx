上下文传播允许您将自定义键值数据从客户端传递到 Workflow，以及从 Workflow 传递到活动和 Child Workflow，而无需通过每个函数签名来线程化值。

常见用例：

- 传播分布式跟踪 ID（例如 OpenTelemetry 跟踪上下文）
- 为多租户应用程序传递租户 ID
- 转发身份验证令牌或请求范围的元数据

每个 SDK 都提供一个 **上下文传播器** 接口，您可以实现该接口来控制注入和提取哪些值。您在客户端注册传播器，SDK 在每个边界自动调用它们。
