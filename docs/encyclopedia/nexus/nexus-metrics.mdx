:::tip SUPPORT, STABILITY, and DEPENDENCY INFO

[Temporal Cloud](/evaluate/development-production-features/release-stages#general-availability) 和 [自托管部署](/cloud/nexus) 的 Temporal Nexus 现在为 [一般可用](/production-deployment/self-hosted-guide/nexus)。

:::


Nexus除了集成的[执行调试](/nexus/execution-debugging)之外，还提供SDK指标、云指标和OSS集群指标。

<a id="sdk-metrics"></a>
## SDK 指标 {#sdk-metrics}

[SDK指标](/references/sdk-metrics) 从 Nexus Worker 发出，包括：

- [Nexus_poll_no_task](/references/sdk-metrics#nexus_poll_no_task)
- [Nexus_task_Schedule_to_start_latency](/references/sdk-metrics#nexus_task_schedule_to_start_latency)
- [Nexus_task_execution_failed Worker](/references/sdk-metrics#nexus_task_execution_failed)
- [Nexus_task_execution_latency](/references/sdk-metrics#nexus_task_execution_latency)
- [Nexus_task_endtoend_latency](/references/sdk-metrics#nexus_task_endtoend_latency)

<a id="cloud-metrics"></a>
## 云指标 {#cloud-metrics}

[云指标](/cloud/metrics/reference) 由 Temporal Cloud 发出，包括：

- 呼叫者 Namespace
  - 响应WorkflowTaskCompleted \- Schedule a Nexus 操作。
- 处理程序 Namespace
  - PollNexusTaskQueue \- 获取要处理的 [Nexus 任务](/tasks#nexus-task)，例如启动 Nexus 操作。
  - RespondNexusTaskCompleted \- 报告 Nexus 任务成功。
  - RespondNexusTaskFailed \- 报告 Nexus 任务失败。

<a id="oss-cluster-metrics"></a>
## OSS集群指标 {#oss-cluster-metrics}

[集群指标](/references/cluster-metrics#nexus-metrics)是从OSS集群发出的，包括：

- 历史服务指标
- 并发限制器指标
- 前端服务指标
