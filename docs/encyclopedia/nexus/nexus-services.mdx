:::tip SUPPORT, STABILITY, and DEPENDENCY INFO

[Temporal Cloud](/evaluate/development-production-features/release-stages#general-availability) 和 [自托管部署](/cloud/nexus) 的 Temporal Nexus 现在为 [一般可用](/production-deployment/self-hosted-guide/nexus)。

:::

[Nexus 服务](/glossary#nexus-service) 被命名为 [Nexus 操作](/nexus/operations) 的集合，提供跨团队边界共享的合同。
[Nexus 端点](/nexus/endpoints) 公开服务供调用者使用。

服务在轮询端点的目标 Task Queue 的 Worker 中注册。
多个服务可以在同一个 Worker 中运行。
服务通常与它们抽象的 Workflow 一起运行，或者使用 [路由器队列模式](/nexus/patterns#router-queue-pattern) 在专用路由器 Worker 中运行。

调用者在执行 Nexus 操作时通过名称引用服务。
