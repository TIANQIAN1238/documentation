在 Temporal 中，超时会检测应用程序故障。
然后，系统可以通过重试自动缓解这些故障。
Workflow 和 Activity 都有专用的超时配置，并且可以使用 RetryPolicy 进行配置。

- [检测 Workflow 故障](/encyclopedia/detecting-workflow-failures)
- [检测 Activity 故障](/encyclopedia/detecting-activity-failures)
- [重试策略](/encyclopedia/retry-policies)
