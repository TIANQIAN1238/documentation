Temporal 提供了许多机制来增强 Workflow 和活动的功能。
这些允许您自定义数据的序列化方式、跨执行边界传播元数据以及注入横切行为（例如跟踪和日志记录）。

- [数据转换](/dataconversion) - 自定义参数和返回值的序列化、压缩或加密方式
- [上下文传播](/encyclopedia/context-propagation) - 跨 Workflow、Activity 和 Child Workflow 边界传递自定义元数据（跟踪 ID、租户 ID、身份验证令牌）
- [拦截器](/encyclopedia/interceptors) - 在 SDK 操作之前和之后添加横切行为（可观察性、授权、标头操作）
- [插件](/encyclopedia/plugins) - 将拦截器、上下文传播器、Data Converter 和内置定义捆绑到可重用包中
