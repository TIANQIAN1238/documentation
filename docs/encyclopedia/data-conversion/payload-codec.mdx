## 什么是有效负载编解码器？ {#payload-codec}

有效负载编解码器将 [有效载荷](/dataconversion#payload) 数组（例如，Workflow 参数列表）转换为另一个有效负载数组。

序列化为有效负载时，首先应用有效负载转换器将对象转换为字节，然后应用编解码器将字节转换为字节。
从有效负载反序列化时，首先应用编解码器以反转效果，然后应用有效负载转换器。

使用自定义有效负载编解码器来转换您的有效负载；例如，对 Workflow Execution 数据实施压缩和/或加密。

### 加密 {#encryption}

在自定义 Data Converter 中使用端到端加密可确保 Temporal Server 处理敏感应用程序数据时的安全。

在自定义负载编解码器中应用您的加密逻辑，并在本地使用它来加密数据。
您维护所有加密密钥，并且 Temporal Server 只能看到加密数据。请参阅 [什么是密钥管理？](/key-management) 了解更多指导。

您的数据仅以未加密的方式存在于客户端以及您控制的主机上执行 Workflow 和活动的 Worker 进程上。有关详细信息，请参见 [保护您的数据](/production-deployment/data-encryption)。
