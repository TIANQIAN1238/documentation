拦截器允许您在 SDK 操作之前和之后添加横切行为，例如启动 Workflow、执行 Activity 或处理 Signal。它们的工作方式类似于中间件：每个拦截器都包装下一个拦截器，形成围绕底层操作执行的链。

常见用例：

- 可观察性（日志记录、指标、跟踪）
- 授权和身份验证检查
- 标头操作（传播元数据）
- 输入/输出验证
