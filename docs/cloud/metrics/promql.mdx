:::danger PromQL endpoint deprecated

PromQL 端点及其 `Temporal_cloud_v0_*` 指标已于 2026 年 4 月 2 日弃用，并且不再接受新用户。
将于 **2026 年 10 月 5 日**对所有用户禁用 PromQL 端点。

迁移到 [OpenMetrics 端点](/cloud/metrics/openmetrics)。
有关完整的 v0 到 v1 指标映射，请参阅 [迁移指南](/cloud/metrics/openmetrics/migration-guide)。

:::

您帐户中所有 Namespace 的指标均可从您的指标端点获取。请记住，您的 Temporal Cloud 指标滞后实时性能大约一分钟。 Temporal Cloud 也仅保留 7 天的原始指标。



为了确保指标的安全性，需要专用于可观察性的 CA 证书。
只有使用该 CA 签名的证书或链接到该 CA 的证书的客户端才可以 Query 指标端点。
有关 Temporal Cloud 中 CA 证书的更多信息，请参阅 [证书要求](/cloud/certificates#certificate-requirements)。
