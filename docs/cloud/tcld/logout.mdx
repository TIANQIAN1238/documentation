`tcld logout` 命令将用户从 Temporal Cloud 中注销。

别名：`lo`

`tcld logout`

以下修饰符控制命令的行为。

<a id="disable-pop-up"></a>
#### --禁用弹出窗口 {#disable-pop-up}

如果设置为 `true`，则禁用浏览器弹出窗口。默认值为 `false`。
