`tcld feature` 命令管理 Temporal Cloud 中的功能。

别名：`f`

- [tcld 功能获取](#get)
- [tcld 功能切换](#toggle)

## 得到 {#get}

`tcld feature get` 命令获取有关您已启用的 Temporal Cloud 功能的信息。

别名：`g`

`tcld feature get`

该命令没有修饰符。

**例子**

`tcld feature get`

以下是示例输出：

```json
[
  {
    "Name": "enable-apikey",
    "Value": true
  }
]
```

<a id="get"></a>
## 切换 {#toggle}

`tcld feature toggle-*` 命令打开或关闭 Temporal Cloud 中的 `*` 功能。

:::note

`*` 符号代表功能的名称。
将 `*` 替换为要切换的可用功能的名称。

:::

别名：`tak`

`tcld feature toggle-*`

该命令没有修饰符。

**例子**

`tcld feature toggle-apikey`

以下是示例输出：

```json
Feature flag enable-apikey is now true
```

:::note

功能 `apikey` 就是一个示例。
Update 用于切换不同功能的功能名称。

:::
