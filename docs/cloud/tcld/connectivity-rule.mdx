`tcld connectivity-rule` 命令管理 Temporal Cloud 中的 [连接规则](/cloud/connectivity#connectivity-rules)。

别名：`cr`

- [tcld 连接规则创建](#create)
- [tcld 连接规则删除](#delete)
- [tcld 连接规则获取](#get)
- [tcld 连接规则列表](#list)

## 创造 {#create}

`tcld connectivity-rule create` 命令创建连接规则。

别名：`c`

<a id="create"></a>
#### --连接 ID {#connection-id}

专用连接的连接 ID。

别名：`ci`

<a id="connection-id"></a>
#### --连接类型 {#connectivity-type}

连接类型，目前仅支持“私有”和“公共”。

别名：`ct`

<a id="connectivity-type"></a>
#### --gcp-项目-id {#gcp-project-id}

连接的 GCP 项目 ID，如果云提供商是“gcp”，则需要该 ID。

别名：`gpi`

#### - 地区 {#region}

连接的区域。

别名：`r`

## 删除 {#delete}

`tcld connectivity-rule delete` 命令删除连接规则。

别名：`d`

<a id="gcp-project-id"></a>
#### --连接规则 ID {#connectivity-rule-id}

连接规则 ID。

别名：`id`

## 得到 {#get}

`tcld connectivity-rule get` 命令获取连接规则。

别名：`g`

<a id="region"></a>
#### --连接规则 ID {#connectivity-rule-id}

连接规则 ID。

别名：`id`

## 列表 {#list}

`tcld connectivity-rule list` 命令列出连接规则。

别名：`l`

<a id="delete"></a>
#### --Namespace {#namespace}

Namespace 托管在 Temporal Cloud 上。

别名：`n`
