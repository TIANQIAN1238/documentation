`tcld request` 命令管理 Temporal Cloud 中的异步请求。

别名：`r`

- [tcld 请求获取](#get)

## 得到 {#get}

`tcld request get` 命令获取 Temporal Cloud 中指定请求的状态。

`tcld request get --request-id <request_id>`

别名：`g`

以下修饰符控制命令的行为。

#### - 要求 {#request}

_所需修饰符_

指定请求标识符。

别名：`-r`

**例子**

```bash
tcld request get --request-id <request_id>
```
