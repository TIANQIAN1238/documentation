Temporal Nexus 现在是 [一般可用](/evaluate/development-production-features/release-stages#general-availability)。
了解为什么应在 [评估指南](/evaluate/nexus) 中使用 Nexus。

:::

Temporal Cloud 中的 Nexus 安全性：

- **[运行时访问控制](/nexus/security#runtime-access-controls)** - 端点白名单限制哪些调用者 Namespace 可以使用端点。参见 [配置访问控制](/nexus/registry#configure-runtime-access-controls)。
- **[安全连接](/nexus/security#secure-connectivity)** - mTLS 用于跨单元和区域的所有 Nexus 通信。端点只能在 Temporal Cloud 帐户内访问。
- **[有效负载加密](/nexus/security#payload-encryption-data-converter)** - Data Converter 与 Workflow 和活动相同，具有三种跨 Namespace 加密方法。
- **[注册表角色和权限](/nexus/registry#roles-and-permissions)** - 控制谁可以查看、创建、编辑和删除端点。
