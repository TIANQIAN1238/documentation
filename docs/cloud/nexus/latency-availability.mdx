:::tip SUPPORT, STABILITY, and DEPENDENCY INFO

Temporal Nexus 现在是 [一般可用](/evaluate/development-production-features/release-stages#general-availability)。
了解为什么应在 [评估指南](/evaluate/nexus) 中使用 Nexus。

:::

Temporal Cloud 中的 Nexus 延迟和可用性：

- **SLO 和 SLA** - Nexus 操作（例如，`RespondWorkflowTaskCompleted`、`PollNexusTaskQueue`、`RespondNexusTaskCompleted` 和 `RespondNexusTaskFailed`）与调用方和处理程序 Namespace 中的其他 Worker 请求具有相同的 [延迟 SLO](/cloud/service-availability#latency) 和 [可用性 SLA](/cloud/sla)。
- **[Nexus 指标](/nexus/metrics)** - 用于监控 Nexus 性能的 SDK 和云延迟指标。
- **跨 Namespace 连接** - 通过全球 mTLS 安全 Envoy 网格进行流量路由。同区域Namespace低延迟；跨区域延迟因提供商而异。参见 [安全连接](/nexus/security#secure-connectivity)。
