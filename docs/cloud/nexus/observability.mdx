Temporal Cloud 中的 Nexus 可观测性：

- **[Nexus 指标](/nexus/metrics)** - Worker 发出的 [SDK指标](/nexus/metrics#sdk-metrics) 和 Temporal Cloud 发出的 [云指标](/nexus/metrics#cloud-metrics)。
- **[执行调试](/nexus/execution-debugging)** - 双向链接、待处理操作、待处理回调以及跨 Namespace 的跟踪。
- **[审计日志记录](/cloud/audit-logs)** - `CreateNexusEndpoint`、`UpdateNexusEndpoint` 和 `DeleteNexusEndpoint` 操作流式传输到您的审核日志集成。
