<a id="start-dev"></a>
## 启动开发 {#start-dev}

在本地系统上运行开发 Temporal Server。

```
+------------------------------------------------------------------------+
| WARNING: The development server is not intended for production use.    |
| It skips certain HTTP security checks to make local use simpler.       |
|                                                                        |
| For production use, see:                                               |
| https://docs.temporal.io/production-deployment                         |
+------------------------------------------------------------------------+
```

查看默认配置的 Web UI：http://localhost:8233

```
temporal server start-dev
```

跨运行添加 Workflow Execution 的持久性：

```
temporal server start-dev \
    --db-filename path-to-your-local-persistent-store
```

从前端gRPC服务设置端口（默认7233）：

```
temporal server start-dev \
    --port 7000
```

使用 Web UI 的自定义端口。默认是gRPC端口（默认7233）
加 1000 (8233)：

```
temporal server start-dev \
    --ui-port 3000
```

使用以下选项更改此命令的行为。您还可以使用适用于所有子命令的任何 [全局标志](#global-flags)。

| 旗帜 | 必需的 | 描述 |
|------|----------|-------------|
| `--db-filename`、`-f` | 不 | **字符串** 持久 Temporal 状态存储的文件路径。默认情况下，当服务器进程终止时，Workflow Execution 会丢失。 |
| `--dynamic-config-value` | 不 | **字符串[]** 使用 `KEY=VALUE` 对的动态配置值。键必须是标识符，值必须是 JSON 值。例如：`YourKey="YourString"` 可以多次传递。 |
| `--headless` | 不 | **布尔** 禁用 Web UI。 |
| `--http-port` | 不 | **int** HTTP API 服务的端口。默认为随机空闲端口。 |
| `--ip` | 不 | **字符串** 前端Service绑定的IP地址。 |
| `--log-config` | 不 | **bool** 将服务器配置记录到 stderr。 |
| `--metrics-port` | 不 | **int** “/metrics”HTTP 端点的端口。默认为随机空闲端口。 |
| `--Namespace`、`-n` | 不 | **字符串[]** Namespace 在启动时创建。 “默认”Namespace 始终是自动创建的。 |
| `--port`、`-p` | 不 | **int** 前端 gRPC 服务的端口。 |
| `--search-attribute` | 不 | **字符串[]** Search Attribute 使用 `KEY=VALUE` 对进行注册。键必须是标识符，值必须是 Search Attribute 类型，它是以下类型之一：Text、Keyword、Int、Double、Bool、Datetime、KeywordList。 |
| `--sqlite-pragma` | 不 | **字符串[]** “PRAGMA=VALUE”格式的 SQLite pragma 语句。 |
| `--ui-asset-path` | 不 | **字符串** UI 自定义资源路径。 |
| `--ui-codec-endpoint` | 不 | **字符串** UI 远程编解码器 HTTP 端点。 |
| `--ui-ip` | 不 | **字符串** 绑定到 Web UI 的 IP 地址。默认与“--ip”值相同。 |
| `--ui-port` | 不 | **int** Web UI 的端口。默认为“--port”值 + 1000。 |
| `--ui-public-path` | 不 | **字符串** Web UI 的公共基本路径。默认为 `/`。 |

| `--client-authority` | 不 | **字符串** Temporal gRPC 客户端：权限伪标头。 |  |

| `--color` | 不 | **字符串枚举** 输出着色。接受的值：始终、从不、自动。 | `auto` |
