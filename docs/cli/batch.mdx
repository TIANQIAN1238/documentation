## 描述 {#describe}

显示正在进行的批处理作业的进度。传递有效的作业 ID 以显示其
信息：

```
temporal batch describe \
    --job-id YourJobId
```

使用以下选项更改此命令的行为。您还可以使用适用于所有子命令的任何 [全局标志](#global-flags)。

| 旗帜 | 必需的 | 描述 |
|------|----------|-------------|
| `--job-id` | 是的 | **字符串** 批处理作业 ID。 |

## 列表 {#list}

返回服务上或单个 Namespace 内的批处理作业列表。对于
例如，列出“YourNamespace”的批处理作业：

```
temporal batch list \
    --namespace YourNamespace
```

使用以下选项更改此命令的行为。您还可以使用适用于所有子命令的任何 [全局标志](#global-flags)。

| 旗帜 | 必需的 | 描述 |
|------|----------|-------------|
| `--limit` | 不 | **int** 要显示的批处理作业的最大数量。 |

## 终止 {#terminate}

使用提供的作业 ID 终止批处理作业。您必须提供理由
终止。该服务将此解释存储为元数据
终止事件供以后参考：

```
temporal batch terminate \
    --job-id YourJobId \
    --reason YourTerminationReason
```

使用以下选项更改此命令的行为。您还可以使用适用于所有子命令的任何 [全局标志](#global-flags)。

| 旗帜 | 必需的 | 描述 |
|------|----------|-------------|
| `--job-id` | 是的 | **字符串** 要终止的作业 ID。 |
| `--reason` | 是的 | **字符串** 终止批处理作业的原因。 |
