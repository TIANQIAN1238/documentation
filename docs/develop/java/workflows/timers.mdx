## 什么是 Timer？ {#timers}

Workflow 可以将耐用的 Timer 设置为固定时间段。
在某些 SDK 中，该函数称为 `sleep()`，而在其他 SDK 中，该函数称为 `Timer()`。

Workflow 可以休眠数月。

Timer 是持久的，因此即使您的 Worker 或 Temporal 服务在时间段结束时关闭，只要您的 Worker 和 Temporal 服务备份，`Workflow.sleep()` 调用就会解析并且您的代码会继续执行。

睡眠是一种轻资源操作：它不会占用进程，并且您可以在单个 Worker 上运行数百万个 Timer。

要在 Java 中设置 Timer，请使用 [`sleep()`](https://www.javadoc.io/doc/io.temporal/temporal-sdk/latest/io/temporal/workflow/Workflow.html#sleep) 并传递您想要在继续​​之前等待的秒数。

```java
sleep(5);
```
