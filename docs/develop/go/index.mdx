<a id="code-samples"></a>
### SDK 特定的代码示例在哪里？ {#code-samples}

- [背景调查申请](https://github.com/temporalio/background-checks)：提供了一个重要的 Temporal
  应用程序实施结合
  [申请文件](https://learn.temporal.io/examples/go/background-checks/)。
- [Go 中的 Hello world 应用程序模板](https://github.com/temporalio/hello-world-project-template-go)：提供
  为用户快速启动开发应用程序。该示例与
  [“你好世界！” Go 从头开始​​教程](https://learn.temporal.io/getting_started/go/hello_world_in_go/)。
- [Go 中的转账应用程序模板](https://github.com/temporalio/money-transfer-project-template-go)：
  为用户提供快速入门的开发应用。它演示了基本的“汇款”Workflow Definition 和
  与
  [在 Go 中运行您的第一个应用程序教程](https://learn.temporal.io/getting_started/go/first_program_in_go/)。
- [Go 中的订阅式 Workflow Definition](https://github.com/temporalio/subscription-workflow-project-template-go)：
  演示一些可以为订阅式业务流程实现的模式。
- [Go 中的电子商务应用示例](https://github.com/temporalio/temporal-ecommerce)：展示每个用户的购物
  购物车风格的 Workflow Definition，包括用于在购物车中添加和删除商品的 API 以及 Web UI。
  该应用程序示例与
  [Go 电子商务教程](https://learn.temporal.io/tutorials/go/build-an-ecommerce-app)。
