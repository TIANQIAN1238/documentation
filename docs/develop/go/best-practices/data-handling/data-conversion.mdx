Temporal SDK 提供了默认的 [Payload Converter](/payload-converter)，可以自定义以将自定义数据类型与 [Payload](/dataconversion#payload) 相互转换。

编码 Payload Converter 的应用顺序取决于传递给 Data Converter 的顺序。
你可以设置多个编码 Payload Converter 来执行转换。
当 Data Converter 收到需要转换的值时，它会依次经过每个 Payload Converter，直到能够处理该数据类型的转换器完成转换为止。

Payload Converter 可以独立于 Payload Codec 进行自定义。

## 使用自定义 Payload Converter {#custom-payload-converter}

使用 [Composite Data Converter](https://pkg.go.dev/go.temporal.io/sdk/converter#CompositeDataConverter) 按指定顺序应用自定义的、类型特定的 Payload Converter。
实现自定义数据处理并不总是需要定义新的 Composite Data Converter。
你可以使用自定义 Codec 覆盖默认 Converter，但对于复杂的 Workflow 逻辑，可能需要使用 Composite Data Converter。

`NewCompositeDataConverter` 从有序的类型特定 Payload Converter 列表创建 `CompositeDataConverter` 的新实例。
Go SDK 中提供了以下类型特定的 Payload Converter，按默认 Data Converter 的应用顺序列出：

- [NewNilPayloadConverter()](https://pkg.go.dev/go.temporal.io/sdk/converter#NilPayloadConverter.ToString)
- [NewByteSlicePayloadConverter()](https://pkg.go.dev/go.temporal.io/sdk/converter#ByteSlicePayloadConverter)
- [NewProtoJSONPayloadConverter()](https://pkg.go.dev/go.temporal.io/sdk/converter#ProtoJSONPayloadConverter)
- [NewProtoPayloadConverter()](https://pkg.go.dev/go.temporal.io/sdk/converter#ProtoPayloadConverter)
- [NewJSONPayloadConverter()](https://pkg.go.dev/go.temporal.io/sdk/converter#JSONPayloadConverter)

Payload Converter 的应用顺序非常重要，因为在序列化过程中，Data Converter 会按照特定顺序依次尝试每个 Payload Converter，直到某个 Payload Converter 返回非 nil 的 Payload 为止。

要设置自定义 Payload Converter，请使用 [`NewCompositeDataConverter`](https://pkg.go.dev/go.temporal.io/sdk/converter#NewCompositeDataConverter) 并在 Client 选项中将其设置为 Data Converter。

- 要用自定义的 `NewCompositeDataConverter` 替换默认 Data Converter，请使用以下代码：

  ```go
  dataConverter := converter.NewCompositeDataConverter(YourCustomPayloadConverter())
  ```

- 要将自定义类型转换添加到默认 Data Converter，请使用以下代码保留默认值，同时在默认 JSON 兜底之前设置你的转换器：

  ```go
  dataConverter := converter.NewCompositeDataConverter(
    converter.NewNilPayloadConverter(),
    converter.NewByteSlicePayloadConverter(),
    converter.NewProtoJSONPayloadConverter(),
    converter.NewProtoPayloadConverter(),
    YourCustomPayloadConverter(),
    converter.NewJSONPayloadConverter(),
  )
  ```
