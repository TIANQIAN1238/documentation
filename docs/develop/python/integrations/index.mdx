以下 AI 框架集成可用于 Temporal Python SDK：

| 框架 | SDK文档 | 集成指南 |
| --- | --- | --- |
| 智囊团 | [Braintrust.dev](https://braintrust.dev/docs) | [指导](./braintrust.mdx) |
| 派丹提克人工智能 | [ai.pydantic.dev](https://ai.pydantic.dev/) | [指导](https://ai.pydantic.dev/durable_execution/temporal/) |
| OpenAI 代理 SDK | [openai.github.io](https://openai.github.io/openai-agents-python/) | [指导](https://github.com/temporalio/sdk-python/blob/main/temporalio/contrib/openai_agents/README.md) |

这些集成基于 Temporal Python SDK 的 [插件系统](/develop/plugins-guide) 构建，您也可以使用它来构建自己的集成。
