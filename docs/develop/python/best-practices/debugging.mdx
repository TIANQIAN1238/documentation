### 在开发环境中调试 {#debug-in-a-development-environment}

**如何使用 Temporal Python SDK 在开发环境中进行调试。**

开发 Workflow 时，您可以使用常规开发工具日志记录和调试器来查看 Workflow 中发生了什么。
除了日志记录和调试器等常规开发工具之外，您还可以使用 [网页用户界面](/web-ui) 或 [Temporal CLI](/cli) 查看 Workflow 中发生的情况。

<a id="debug-in-a-production-environment"></a>
### 如何在生产环境中调试 {#debug-in-a-production-environment}

**如何使用 Temporal Python SDK 在生产环境中进行调试。**

您可以使用以下方法调试生产 Workflow：

- [网页用户界面](/web-ui)
- [Temporal CLI](/cli)
- [重播](/develop/python/best-practices/testing-suite#replay)
- [追踪](/develop/python/platform/observability#tracing)
- [记录](/develop/python/platform/observability#logging)

您可以使用指标和 [Worker 性能指南](/develop/worker-performance) 来调试和调整 Worker 性能。
有关更多信息，请参阅 [可观察性 ▶️ 指标](/develop/python/platform/observability#metrics) 设置 SDK 指标。

使用 [云指标](/cloud/metrics/) 或 [自托管服务器指标](/self-hosted-guide/production-checklist#scaling-and-metrics) 调试服务器性能。
