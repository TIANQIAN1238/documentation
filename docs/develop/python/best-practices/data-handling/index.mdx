所有发送到 Temporal Service 以及从 Temporal Service 接收的数据都会经过 **Data Converter**。Data Converter 包含三个
处理不同关注点的层：

在这三层中，只有 PayloadConverter 是必需的。Temporal 使用默认的 PayloadConverter 处理 JSON 序列化。PayloadCodec 和 ExternalStorage 层是可选的。只有当你的应用程序需要非 JSON 类型、加密或 payload 卸载时，才需要自定义这些层。

|                           | [PayloadConverter](/develop/python/data-handling/data-conversion) | [PayloadCodec](/develop/python/data-handling/data-encryption) | [ExternalStorage](/develop/python/data-handling/external-storage) |
| ------------------------- | ----------------------------------------------------------------- | ------------------------------------------------------------- | ---------------------------------------------------------------------- |
| **用途**                  | 将应用程序数据序列化为字节                                         | 转换已编码的 payload（加密、压缩）                             | 将大型 payload 卸载到外部存储                                          |
| **默认值**                | JSON 序列化                                                       | 无（直接透传）                                                 | 无（所有 payload 均存储在 Workflow History 中）                        |

如需更深入的概念解释，请参阅 [Data Conversion 百科全书](/dataconversion) 和[外部存储](/external-storage)。
