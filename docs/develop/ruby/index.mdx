## 安装并开始 {#install-and-get-started}

您可以在 [快速入门](/develop/ruby/set-up-local-ruby) 中找到 Ruby SDK 的详细安装说明。

还有一个关于如何使用 Temporal 原语（活动、Workflow 和 Worker）来构建和运行 Temporal 应用程序的简短演练，以帮助您启动和运行。

设置本地 Temporal 服务后，请使用以下资源继续构建：

- [Workflow 基础知识](/develop/ruby/workflows/basics)

- [启动 Activity 执行](/develop/ruby/activities/execution)

从那里，您可以更深入地研究任何 Temporal 原语，开始构建适合您的用例的 Workflow。
