## TypeScript 中的 Linting 和类型 {#linting-and-types}

如果您使用 `@Temporalio/create` 开始您的项目，那么您已经拥有我们推荐的 TypeScript 和 ESLint
配置。

如果您将 Temporal 增量添加到现有应用程序中，我们建议您设置 linting 和类型，因为它们有助于在将错误交付到生产之前捕获错误，并且可以改善您的开发反馈循环。查看我们推荐的 [.eslintrc](https://github.com/temporalio/samples-typescript/blob/main/.shared/.eslintrc.js) 文件并进行调整以满足您的需求。
