Temporal 旨在使开发分布式应用程序成为一种愉快的体验。
开发人员可以从一种清晰的方法中受益，该方法可以将代码和 Visibility 构建到应用程序的状态中。
应用程序受益于容错和执行保证。
数千家各种规模的公司正在利用 Temporal 的功能来处理关键任务和标准工作负载。
