Temporal 的可观察性功能可帮助您实时跟踪 Workflow 的状态，提供用于详细指标、跟踪、综合日志记录和 Visibility 应用程序状态的工具。

监控性能、跟踪 Activity 和 Workflow Execution、调试和过滤 Workflow Execution，以更深入地了解您的 Workflow。

**Temporal 可观测性和 Visibility 的关键组件**

- **指标**：详细的性能指标，用于跟踪 Temporal 服务和 Workflow 的运行状况和效率。
- **跟踪**：对 Workflow 和 Activity 执行进行端到端跟踪，以了解操作的流程和时间安排。
- **日志记录**：用于调试和审计目的的全面日志记录功能。
- **Search Attributes**：可用于增强可搜索性并为 Workflow Executions 提供附加上下文的自定义属性。
- **Web UI**：用户友好的界面，用于可视化 Workflow 和 Temporal 服务状态并与之交互。

**Temporal 的可观测性和 Visibility 功能的优势**

- **实时监控**：跟踪 Workflow 执行时的状态和进度。
- **性能优化**：识别瓶颈并优化您的 Workflow 和 Activity 实施。
- **有效调试**：快速定位和诊断 Temporal 应用程序中的问题。
- **合规性和审计**：维护所有 Workflow Execution 的详细记录，以用于合规性和审计目的。
- **操作洞察**：深入了解应用程序的行为和使用模式。
- **可扩展性管理**：有效监控和管理 Temporal 服务的可扩展性。
