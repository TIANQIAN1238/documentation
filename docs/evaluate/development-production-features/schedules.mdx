Temporal Schedules 是一项允许您在指定时间或间隔使用“Schedule”Temporal Workflow 的功能，并根据高峰使用情况进行调整。

它提供了一种灵活的方式来自动化和管理您的 Temporal Workflow，确保您的业务流程平稳高效地运行，尤其是在处理时间敏感的任务时。

1. **自动化重复任务：**
   Schedule 可自动执行重复任务，减少人工干预并确保业务流程的及时执行。
2. **增强的 Workflow 控制和可观察性：**
   完全控制您的自动化流程。使用 Schedule，您可以创建、回填、删除、描述、列出、暂停、触发和 Update Workflow Execution。
3. **灵活的时间安排：**
   Schedule Workflow Execution 定期运行或在未来的特定时间运行，确保它们在需要时精确执行。
4. **可靠且可扩展：**
   Temporal Schedule 专为可靠性和可扩展性而设计，可处理分布式系统的复杂性，同时确保您的 Workflow 按预期运行，即使在出现故障时也是如此。
5. **消除外部依赖：**
   Schedule 无需集成外部调度系统。
