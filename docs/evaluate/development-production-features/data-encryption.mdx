Temporal 中的 Data Converter 是 SDK 组件，用于处理 Temporal CLIent 发送和接收的数据的序列化和编码。
Workflow 输入和输出需要序列化和反序列化，以便它们可以作为 JSON 发送到 Temporal 服务。

Temporal 提供其自己的默认 Data Converter 逻辑，如果有效负载包含纯文本或 JSON 数据，则该逻辑对用户来说并不明显。
为了增强安全性，您可以使用 Codec Server 实施您自己的加密标准。
Temporal 的数据加密功能可确保您的 Workflow 的安全性和机密性，并在不影响性能的情况下提供保护。
