在 Temporal 的上下文中，您可以创建以下类型的自动化测试：

1. 端到端：运行 Temporal Server 和 Worker 及其所有 Workflow 和活动；从客户端启动 Workflow 并与之交互。
2. 集成：介于端到端和单元测试之间的任何内容。
   使用模拟上下文和其他 SDK 导入（通常还有网络请求）运行活动。
   使用模拟活动运行 Worker，并使用客户端启动 Workflow。
   使用模拟 SDK 导入运行 Workflow。
3. 单元：运行一段 Workflow 或 Activity 代码并模拟它调用的任何代码。
