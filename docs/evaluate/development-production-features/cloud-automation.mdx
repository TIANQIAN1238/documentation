Temporal Cloud 自动化改变了您管理和扩展云基础设施的方式。
其功能使您能够自动执行关键任务，例如用户和 Namespace 管理、mTLS 证书轮换和访问控制，从而确保安全性和运营效率。
云自动化提供跨所有接口的安全身份验证，减少错误并增强安全性。

**主要特点：**

- [安全 API 密钥](https://docs.temporal.io/cloud/api-keys)：使用 Temporal Cloud API 密钥安全地管理资源。
- [Temporal Cloud CLI (tcld)](https://docs.temporal.io/cloud/tcld)：直接从命令行自动执行操作。
- [云 Terraform 提供商](https://docs.temporal.io/cloud/terraform-provider#prerequisites)：通过基础设施即代码轻松扩展。
