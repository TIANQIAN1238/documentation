`tctl dataconverter` 命令启用自定义 [Data Converter](/dataconversion) 操作。

- [tctl 数据转换器网络](#web)

<a id="web"></a>
## 网络 {#web}

`tctl dataconverter web` 命令指定与 Temporal Web 一起使用的自定义 [Data Converter](/dataconversion) 的 WebSocket URL。

`tctl dataconverter web --web_ui_url <url>`

以下修饰符控制命令的行为。

### - 港口 {#port}

指定自定义 [Data Converter](/dataconversion) 的 WebSocket URL 的端口。
默认值为 0。

**例子**

```bash
tctl dataconverter web --web_ui_url <url> --port <value>
```

<a id="port"></a>
### --web_ui_url {#webuiurl}

_所需修饰符_

指定自定义 [Data Converter](/dataconversion) 的 WebSocket URL。

**例子**

```bash
tctl dataconverter web --web_ui_url <url>
```
