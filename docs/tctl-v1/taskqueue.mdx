`tctl taskqueue` 命令启用 [Task Queue](/task-queue) 操作。

别名：`t`

- [tctl任务队列描述](#describe)
- [tctl 任务队列列表分区](#list-partition)

## 描述 {#describe}

`tctl taskqueue describe`命令描述了[Task Queue](/task-queue)的轮询器信息。

`tctl taskqueue describe <modifiers> <value>`

以下修饰符控制命令的行为。

<a id="describe"></a>
### --任务队列 {#taskqueue}

_所需修饰符_

指定 [Task Queue](/task-queue)。

别名：`--t`

**例子**

```bash
tctl taskqueue describe --taskqueue <value>
```

<a id="taskqueue"></a>
### --任务队列类型 {#taskqueuetype}

指定 [Task Queue](/task-queue) 的类型。
类型可以是 `Workflow` 或 `Activity`。
默认值为 `Workflow`。

**例子**

```bash
tctl taskqueue describe --taskqueue <value> --taskqueuetype <type>
```

<a id="taskqueuetype"></a>
## 列表分区 {#list-partition}

`tctl taskqueue list-partition` 命令列出 [Task Queue](/task-queue) 的分区以及分区的主机名。

`tctl taskqueue list-partition --taskqueue <value>`

以下修饰符控制命令的行为。

<a id="list-partition"></a>
### --任务队列 {#taskqueue}

_所需修饰符_

指定 [Task Queue](/task-queue) 描述。

别名：`--t`

**例子**

```bash
tctl taskqueue list-partition --taskqueue <value>
```
