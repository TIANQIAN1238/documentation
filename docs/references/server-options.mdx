您可以通过包含服务器包 `go.Temporal.io/server/temporal` 并使用它来创建和启动 Temporal Server，将 [Temporal Server](/temporal-service/temporal-server) 作为 Go 应用程序运行。

Temporal Server 服务可以通过多种方式运行。
我们建议在有限的情况下使用这种方法。

```go
s, err := temporal.NewServer()
if err != nil {
	log.Fatal(err)
}
err = s.Start()
if err != nil{
	log.Fatal(err)
}
```

`NewServer()` 接受函数作为参数。
每个函数都会返回一个应用于实例的 `ServerOption`。
参数参考源代码在这里：https://github.com/temporalio/temporal/blob/main/temporal/server_option.go

### WithConfig

要启动 Temporal Server，需要配置文件。服务器自动搜索该配置
启动时位于默认位置./config/development.yaml。如果您需要使用自定义配置，您可以
通过服务器的配置选项指定它。有关配置参数的全面详细信息和
结构，请参考[官方配置文档](https://pkg.go.dev/go.temporal.io/server/common/config)。

```go
s, err := temporal.NewServer(
	temporal.WithConfig(cfg),
)
```

### WithConfigLoader

从文件加载自定义配置。

```go
s, err := temporal.NewServer(
	temporal.WithConfigLoader(configDir, env, zone),
)
```

<a id="forservices"></a>
### 对于服务 {#forservices}

设置所有有效 Temporal 服务的列表。
可以使用 `go.Temporal.io/server/temporal` 包中的默认值。

```go
s, err := temporal.NewServer(
	temporal.ForServices(temporal.Services),
)
```

<a id="interrupton"></a>
### 中断 {#interrupton}

此选项提供一个通道，可从该通道中断 Signal 上的服务器。

- 如果 `Temporal.InterruptOn()` 没有通过，`server.Start()` 永远不会被阻塞，你需要在某个地方调用 `server.Stop()`。
- 如果 `Temporal.InterruptOn(nil)` 被传递，则 `server.Start()` 永远阻塞，直到进程被终止。
- 如果 `Temporal.InterruptOn(Temporal.InterruptCh())` 被传递，`server.Start()` 会阻塞，直到您使用 Ctrl+C，然后正常关闭服务器。
- 如果 `Temporal.Interrupt(someCustomChan)` 被传递，则 `server.Start()` 会阻塞，直到 Signal 发送到 `someCustomChan`。

```go
s, err := temporal.NewServer(
	temporal.InterruptOn(temporal.InterruptCh()),
)
```

### WithAuthorizer

设置低级别 [授权机制](/self-hosted-guide/security#authorizer-plugin) 以确定是允许还是拒绝入站 API 调用。

```go
s, err := temporal.NewServer(
	temporal.WithAuthorizer(myAuthorizer),
)
```

### WithTLSConfigFactory

覆盖默认的 TLS 配置提供程序。
`TLSConfigProvider` 在 `go.Temporal.io/server/common/rpc/encryption` 包中定义。

```go
s, err := temporal.NewServer(
	temporal.WithTLSConfigFactory(yourTLSConfigProvider),
)
```

### WithClaimMapper

将 [角色映射机制](/self-hosted-guide/security#claim-mapper) 配置为 `Claims` 以进行授权。

```go
s, err := temporal.NewServer(
  temporal.WithClaimMapper(func(cfg *config.Config) authorization.ClaimMapper {
  		logger := getYourLogger() // Replace with how you retrieve or initialize your logger
		return authorization.NewDefaultJWTClaimMapper(
			authorization.NewDefaultTokenKeyProvider(cfg, logger),
			cfg
		)
	}),
)
```

### WithCustomMetricsReporter

设置自定义计数指标报告器。

```go
s, err := temporal.NewServer(
	temporal.WithCustomMetricsReporter(myReporter),
)
```

您可以查看 [自定义记者的 Uber 统计文档](https://github.com/uber-go/tally#report-your-metrics) 并查看 [Datadog `dogstatsd` 格式的报告器](https://github.com/temporalio/temporal/pull/998#issuecomment-857884983) 的社区实现。
