虽然通常通过以下方式与 [Temporal Server](/temporal-service/temporal-server) 交互是最简单的
[客户端SDK](/encyclopedia/temporal-sdks#temporal-client) 或 [Temporal CLI](https://docs.temporal.io/cli)，您可以
还可以使用其 gRPC API。

我们的客户端和 Worker SDK 使用 gRPC API。 API 参考位于此处：

[`api-docs.Temporal.io`](https://api-docs.temporal.io/)

<a id="use-with-code"></a>
## 与代码一起使用 {#use-with-code}

通常，您通过 `client.Workflow.start()` 等高级方法与 API 进行交互。然而，客户端 SDK 也
公开底层 gRPC 服务。例如，TypeScript SDK 具有：

- Workflow服务：
  [`Client.connection.WorkflowService`](https://typescript.temporal.io/api/classes/client.Connection#workflowservice)
- 操作员服务：
  [`Client.connection.operatorService`](https://typescript.temporal.io/api/classes/client.Connection/#operatorservice)
- 健康服务：
  [`Client.connection.healthService`](https://typescript.temporal.io/api/classes/client.Connection/#healthservice)

如果您不使用 SDK 客户端（罕见），则可以通过以下方式生成 gRPC 客户端存根：

- 克隆 [`Temporalio/api`](https://github.com/temporalio/api)（带有 Protobuf 文件的存储库）
- 在[你的语言](https://grpc.io/docs/languages/)中生成代码

<a id="use-manually"></a>
## 手动使用 {#use-manually}

要通过命令行或 GUI 手动使用 Query API，首先：

- 如果您还没有要连接的服务器，请运行 [`Temporal Server start-dev`](/cli/server#start-dev)
- 克隆这个仓库：

```shell
git clone https://github.com/temporalio/api.git
cd api
```

<a id="with-command-line"></a>
### 使用命令行 {#with-command-line}

安装 [`evans`](https://github.com/ktr0731/evans#installation)。

```shell
cd /path/to/api
evans --proto temporal/api/workflowservice/v1/service.proto --port 7233
```

要连接到 Temporal Cloud，请设置主机、证书、证书密钥和 TLS 标志：

```shell
evans --proto temporal/api/workflowservice/v1/service.proto --host devrel.a2dd6.tmprl.cloud --port 7233 --tls --cert /Users/me/certs/temporal.pem --certkey /Users/me/certs/temporal.key
```

进入 evans 提示符后，您可以运行 `help`、`show service` 等命令来列出可用的方法，以及
`call ListWorkflowExecutions`。

<a id="with-a-gui"></a>
### 使用 GUI {#with-a-gui}

- 安装 [布卢姆RPC](https://github.com/bloomrpc/bloomrpc#installation)。
- 打开应用程序
- Select "Import Paths" button on the top-left and enter the path to the cloned repo: `/path/to/api`
- 选择“导入原型”+按钮并选择此文件：

```shell
/path/to/api/temporal/api/workflowservice/v1/service.proto
```

- 方法列表应出现在侧栏中。选择一个。
- 在左侧窗格中编辑 JSON。
- 点击 `Cmd/Ctrl-Enter` 或单击播放按钮即可从右侧服务器获取响应。
