＃＃ 描述

显示正在进行的批处理作业的进度。传递有效的作业 ID 以显示其
信息：

```
temporal batch describe \
    --job-id YourJobId
```

使用以下选项更改此命令的行为。您还可以使用任何适用于所有子命令的[全局标志](#global-flags)。

|旗帜|必填 |描述 |
|------|----------|-------------|

## 终止

使用提供的作业 ID 终止批处理作业。您必须提供理由
终止。该服务将此解释存储为元数据
终止事件供以后参考：

```
temporal batch terminate \
    --job-id YourJobId \
    --reason YourTerminationReason
```

使用以下选项更改此命令的行为。您还可以使用任何适用于所有子命令的[全局标志](#global-flags)。

|旗帜|必填 |描述 |
|------|----------|-------------|
| `--job-id` |是的 | **字符串** 要终止的作业 ID。 |
| `--原因` |是的 | **字符串** 终止批处理作业的原因。 |
