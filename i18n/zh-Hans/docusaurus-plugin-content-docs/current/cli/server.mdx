本页提供“Temporal”CLI“server”命令的参考。适用于每个子命令的标志显示在子命令标题内的表格中。请参阅 [全局标志](#global-flags) 了解可与每个子命令一起使用的标志。

## 启动开发

在本地系统上运行开发 Temporal Server。

```
+------------------------------------------------------------------------+
| WARNING: The development server is not intended for production use.    |
| It skips certain HTTP security checks to make local use simpler.       |
|                                                                        |
| For production use, see:                                               |
| https://docs.temporal.io/production-deployment                         |
+------------------------------------------------------------------------+
```

查看默认配置的 Web UI：http://localhost:8233

```
temporal server start-dev
```

添加跨运行的 Workflow Execution 持久性：

```
temporal server start-dev \
    --db-filename path-to-your-local-persistent-store
```

设置前端gRPC服务的端口（默认7233）：

```
temporal server start-dev \
    --port 7000
```

将复杂的流程分解为父进程和 XPROT011X（或 XPROT039X）是一种常见且通常合适的模式，但 APS 成本随着深度和扇出而急剧增加。

```
temporal server start-dev \
    --ui-port 3000
```

使用以下选项更改此命令的行为。您还可以使用任何适用于所有子命令的[全局标志](#global-flags)。

|旗帜|必填 |描述 |
|------|----------|-------------|
| `--db-文件名`, `-f` |没有 | **字符串** 持久 Temporal 状态存储的文件路径。默认情况下，当服务器进程终止时，Workflow Execution 会丢失。 |
| `--动态配置值` |没有 | **string[]** 使用“KEY=VALUE”对的动态配置值。键必须是标识符，值必须是 JSON 值。例如： `YourKey="YourString"` 可以多次传递。 |
| `--无头` |没有 | **布尔** 禁用 Web UI。 |
| `--http-port` |没有 | **int** HTTP API 服务的端口。默认为随机空闲端口。 |
| `--ip` |没有 | **字符串** 前端Service绑定的IP地址。 |
| `--log-config` |没有 | **bool** 将服务器配置记录到 stderr。 |
| `--metrics-port` |没有 | **int** “/metrics”HTTP 端点的端口。默认为随机空闲端口。 |
| `--Namespace`、`-n` |没有 | **字符串[]** Namespace 在启动时创建。 “默认”Namespace 始终自动创建。 |
| `--port`、`-p` |没有 | **int** 前端 gRPC 服务的端口。 |
| `--搜索属性` |没有 | **string[]** Search Attributes 使用“KEY=VALUE”对进行注册。键必须是标识符，值必须是 Search Attribute 类型，即以下类型之一：Text、Keyword、Int、Double、Bool、Datetime、KeywordList。 |
| `--sqlite-pragma` |没有 | **字符串[]** “PRAGMA=VALUE”格式的 SQLite pragma 语句。 |
| `--ui-资产路径` |没有 | **字符串** UI 自定义资源路径。 |
| `--ui-codec-endpoint` |没有 | **字符串** UI 远程编解码器 HTTP 端点。 |
| `--ui-ip` |没有 | **字符串** 绑定到 Web UI 的 IP 地址。默认与“--ip”值相同。 |
| `--ui-port` |没有 | **int** Web UI 的端口。默认为“--port”值 + 1000。
| `--ui-public-path` |没有 | **字符串** Web UI 的公共基本路径。默认为`/`。 |
