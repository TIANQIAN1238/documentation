### 全局修饰符

#### --自动确认

自动确认所有提示。

您可以通过设置 AUTO_CONFIRM 环境变量来指定此修饰符的值。
默认值为“假”。

## 如何安装 tcld {#install-tcld}

您可以通过两种方式安装[tcld](/cloud/tcld)。

### 使用 Homebrew 安装 tcld

```bash
brew install temporalio/brew/tcld
```

### 从源代码构建 tcld

1. 确认您已安装 Go 1.18 或更高版本。

   ```bash
   go version
   ```

如果未安装 Go 1.18 或更高版本，请按照 Go 网站上的[下载并安装](https://go.dev/doc/install) 说明进行操作。

1. 克隆 tcld 存储库并运行 make。

   ```bash
   git clone https://github.com/temporalio/tcld.git
   cd tcld
   make
   ```

1. 将 tcld 可执行文件复制到 PATH 环境变量中出现的任何目录，例如“/usr/local/bin”。

   ```bash
   cp tcld /usr/local/bin/tcld
   ```

1. 验证 tcld 是否已安装。

   ```bash
   tcld version
   ```
