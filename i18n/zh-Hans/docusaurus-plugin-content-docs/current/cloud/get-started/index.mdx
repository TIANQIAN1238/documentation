## 注册 Temporal Cloud

要创建 Temporal Cloud 帐户，您可以：

- [直接]注册(https://Temporal.io/get-cloud)；或
- 在 AWS Marketplace 订阅
  [Temporal Cloud 即用即付](https://aws.amazon.com/marketplace/pp/prodview-xx2x66m6fp2lo)。通过以下方式注册
  AWS Marketplace 类似于直接在 Temporal Cloud 网站上注册，但计费通过您的 AWS
  帐户。
- 要在 Google Cloud Marketplace 上购买 Temporal Cloud，请通过 sales@Temporal.io 联系我们的团队。

有关 Temporal Cloud 定价的信息，请参阅我们的[定价页面](/cloud/pricing)。

## 创建 Namespace

请参阅[管理 Namespaces](/cloud/Namespaces#create-a-Namespace) 创建您的第一个 Namespace。

Temporal Cloud 支持 [API 密钥](/cloud/api-keys) 或 [mTLS](/cloud/certificates) 身份验证
Namespace。如果您不确定使用哪个，我们建议使用 [API 密钥](/cloud/api-keys)，因为它们更容易
大多数团队的管理和轮换。如果您的组织已经拥有私钥基础设施 (PKI) 并且熟悉
证书管理，那么[mTLS](/cloud/certificates)是一个很好的选择。

## 邀请你的团队

请参阅[管理用户](/cloud/users) 添加其他用户并为其分配角色。

您还可以使用[服务帐户](/cloud/service-accounts)来表示机器身份。

由于您在注册时创建了帐户，因此您的电子邮件地址是第一个
您帐户的[帐户所有者](/cloud/manage-access/roles-and-permissions#account-level-roles)。
