---

了解如何零停机时间迁移 Temporal Workflow：

- [自动迁移](/cloud/migrate/automated) - 此过程可实现从自托管 Temporal 实例到 Temporal Cloud 的无缝过渡。
- [手动迁移](/cloud/migrate/manual) - 此过程允许通过更新客户端从自托管 Temporal 实例转换到 Temporal Cloud
  和 Workflow，以利用 Temporal Cloud 中的新资源。
- [在区域之间迁移](/cloud/migrate/migrate-within-cloud) - 此过程允许您在区域或提供商之间迁移 Temporal Cloud Namespace。
