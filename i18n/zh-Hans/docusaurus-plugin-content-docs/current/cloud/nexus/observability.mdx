Temporal Cloud 中的 Nexus 可观测性：

- **[Nexus 指标](/Nexus/metrics)** - Worker 发出的 [SDK 指标](/Nexus/metrics#sdk-metrics) 和 Temporal Cloud 发出的[云指标](/Nexus/metrics#cloud-metrics)。
- **[执行调试](/Nexus/execution-debugging)** - 双向链接、挂起操作、挂起回调以及 Namespace 之间的跟踪。
- **[审核日志记录](/cloud/audit-logs)** - `CreateNexusEndpoint`、`UpdateNexusEndpoint` 和 `DeleteNexusEndpoint` 操作流式传输到您的审核日志集成。
