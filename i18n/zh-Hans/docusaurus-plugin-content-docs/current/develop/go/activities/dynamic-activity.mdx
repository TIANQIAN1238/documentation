## 设置动态 Activity {#set-a-dynamic-Activity}

Temporal 中的动态 Activity 是一个 Activity，如果没有注册其他同名的 Activity，则在运行时动态调用该 Activity。
可以使用“Worker.RegisterDynamicActivity()”将 Activity 注册为动态。
您必须先向 Worker 注册 Activity，然后才能调用它。
Worker 上只能存在一个 Dynamic Activity。

然后，Activity Definition 必须接受“converter.EncodedValues”类型的单个参数。
此代码片段取自[samples-go 中的动态 Workflow 示例](https://github.com/Temporalio/samples-go/tree/main/dynamic-Workflows)。

```go
func DynamicActivity(ctx context.Context, args converter.EncodedValues) (string, error) {
	var arg1, arg2 string
	err := args.Get(&arg1, &arg2)
	if err != nil {
		return "", fmt.Errorf("failed to decode arguments: %w", err)
	}

	info := activity.GetInfo(ctx)
	result := fmt.Sprintf("%s - %s - %s", info.WorkflowType.Name, arg1, arg2)

	return result, nil
}
```
