以下 AI 框架集成可用于 Temporal Python SDK：

|框架| SDK 文档 |集成指南|
| --- | --- | --- |
|智囊团| [braintrust.dev](https://braintrust.dev/docs) | [指南](./braintrust.mdx) |
|派丹蒂克人工智能 | [ai.pydantic.dev](https://ai.pydantic.dev/) | [指南](https://ai.pydantic.dev/durable_execution/Temporal/) |
| OpenAI 代理 SDK | [openai.github.io](https://openai.github.io/openai-agents-python/) | [指南](https://github.com/Temporalio/sdk-python/blob/main/Temporalio/contrib/openai_agents/README.md) |

这些集成构建在 Temporal Python SDK 的 [插件系统](/develop/plugins-guide) 上，您也可以使用它来构建自己的集成。
