## 安装并开始

您可以在[快速入门](/develop/dotnet/set-up-your-local-dotnet)中找到 .NET SDK 的详细安装说明。

还有关于如何使用 Temporal 原语（活动、Workflow 和 Worker）构建和运行 Temporal 应用程序的简短演练，以便您启动和运行。

设置本地 Temporal 服务后，继续使用以下资源进行构建：

- [Workflow 基础知识](/develop/dotnet/Workflows/basics)
- [Activity 基础知识](/develop/dotnet/activities/basics)
- [启动 Activity 执行](/develop/dotnet/activities/execution)
- [运行 Worker 进程](/develop/dotnet/Workers/run-Worker-process)

从那里，您可以更深入地研究任何 Temporal 原语，开始构建适合您的用例的 Workflow。
